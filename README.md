# Caramel

## 项目介绍

### 关于名称

**Caramel** - Carefree Framework Library

`美音` /'kærəm(ə)l/

`英音` /'kærəmɛl/

`译意` 焦糖；焦糖糖果；饴糖

`释义` `Ca`refree F`rame`work `L`ibrary 一套可以让你更加轻松地对各种开发框架进行配置的工具库
