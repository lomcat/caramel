# Caramel

## 项目介绍

### 关于名称

**Caramel** - Carefree Framework Library

`发音` /'kærəmɛl/、/'kærəm(ə)l/

`译意` 焦糖；焦糖糖果；饴糖

`释义` `Ca`refree F`rame`work `L`ibrary 一套可以让你更加轻松地使用和配置各种开发框架的工具库
